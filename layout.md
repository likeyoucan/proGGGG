1. Экраны и навигация
------------------------------------------------------------
============================================================

------------------------------
Preloader
------------------------------

- Дисклеймер
- Экран студии
  - название студии
  - изображение / логотип студии
- Загрузка
  - индикатор загрузки

------------------------------
Главное меню
------------------------------

- Продолжить
  - активно, если есть сохранения

- Новая игра
  - создание имени персонажа
  - дополнительные параметры (если нужны)

- Загрузить
  - активно, если есть сохранения
  - список сохранений:
    - название
    - дата
    - время
  - кнопки:
    - Назад
    - Загрузить
      - активна, если выбрано сохранение
  - процесс загрузки:
    - подсказки
    - индикатор
    - фоновые изображения

- Достижения
  - опционально

- Авторы

- Настройки
  - переход на экран настроек

- Выход

- Политика конфиденциальности
  - располагается в нижней части экрана

------------------------------
Экран настроек
------------------------------

- Аудио
  - звук (переключатель)
  - общая громкость
  - громкость музыки
  - громкость эффектов

- Вибрация
  - вибрация (переключатель)
  - уровень вибрации

- Графика
  - качество графики:
    - низкое
    - среднее
    - высокое
    - максимальное
  - яркость
  - контраст
  - коррекция цветов

- Контент и монетизация
  - скрыть NSFW (переключатель)
  - отключить рекламу (переключатель)

- Назад 

------------------------------
Экран игры
------------------------------

- Верхняя левая часть
  - аватар
  - имя игрока
  - энергия
  - попытки
  - дополнительные опции

- Верхняя правая часть
  - инвентарь
  - карта сюжета
  - меню паузы
  - покупка Premium
    - если включена разработчиком

- Нижняя часть
  - область сюжета
  - область действий
  - список вариантов ответа

- Основная сцена
  - 1 слой: фон
  - 2 слой: персонажи и предметы
  - 3 слой: эффекты

------------------------------
Меню паузы
------------------------------

- Новая игра
  - с предупреждением
- Сохранить
- Загрузить
- В главное меню
- Выход

2. Игровая логика
------------------------------------------------------------
============================================================

------------------------------
Структура сюжета
------------------------------

- Игра состоит из Глав (Chapters)
  - каждая глава имеет:
    - порядковый номер
    - название

- Главы состоят из Сцен / Узлов (Scenes / Nodes)
  - каждая сцена имеет:
    - ID
    - название
    - контент

- Сцены связаны через Варианты выбора (Choices / Edges)
  - каждый вариант имеет:
    - текст
    - ссылку на следующую сцену
    - переход внутри главы или в другую главу

- Разработчик может задавать эффекты перехода
  между сценами в зависимости от выбора игрока

------------------------------
Визуальная система сцены
------------------------------

- Background (фон)
  - статичное изображение
  - серия изображений
  - слайдшоу с плавным переходом
  - зацикленное видео

- Midground (персонажи / объекты)
  - располагается поверх фона

- Foreground (эффекты)
  - наложения
  - атмосферные эффекты
  - дождь
  - блики
  - прочие визуальные эффекты поверх сцены

- Поддержка качества медиа
  - для каждого медиафайла должна быть возможность
    хранить пути к версиям разного качества:
    - Low
    - Medium
    - High
    - Max

- Игра должна загружать нужный файл
  в зависимости от выбранных настроек графики

------------------------------
Меню игры
- продолжить
- новая игра
- загрузить
- настройки
- выход в главное меню
- выход из игры

Игровые ресурсы
------------------------------

- Энергия
  - тратится на прохождение
  - если энергия закончилась, игрок не может выбрать действие

  - ограничивают количество попыток на выбор
    "правильного" варианта ответа

- Инвентарь
  - хранит предметы, полученные по ходу сюжета

3. Системная логика настроек
------------------------------------------------------------
============================================================

------------------------------
Аудио
------------------------------

- Для звука, музыки и SFX используются:
  - тумблер Вкл / Выкл
  - ползунок громкости 0–100%

- Правила:
  - если тумблер выключен — ползунок устанавливается в 0
  - если ползунок установлен в 0 — тумблер выключается
  - если ползунок поднят выше 0 — тумблер включается
  - если тумблер включают после выключения —
    ползунок возвращается на предыдущее значение

------------------------------
Вибрация
------------------------------

- Для вибрации используются:
  - тумблер Вкл / Выкл
  - ползунок силы вибрации

- Правила:
  - если тумблер выключен — ползунок устанавливается в 0
  - если ползунок установлен в 0 — тумблер выключается
  - если ползунок поднят выше 0 — тумблер включается
  - если тумблер включают после выключения —
    ползунок возвращается на предыдущее значение

------------------------------------------------------------
4. Монетизация и реклама
------------------------------------------------------------
============================================================

------------------------------
Premium
------------------------------

- Глобальный флаг:
  - куплена / не куплена

- Если Premium куплена:
  - скрывается вся реклама
  (в настройах опция включения рекламы остается)
  - энергия становится бесконечной

- Если игрок пытается отключить рекламу в настройках:
  - вызывается окно покупки Premium

------------------------------
Interstitial Ads
------------------------------

- Разработчик может назначать показ полноэкранной рекламы:
  - при переходе между главами
  - при переходе между сценами
  - при выходе в главное меню

- Настройка задаётся в свойствах сцены
  или перехода

------------------------------
Rewarded Ads
------------------------------

- Игрок может смотреть рекламу за вознаграждение для:
  - восстановления энергии
  - получения подсказки
  - покупки правильного ответа
